# App Changelog - 2026-06

## 2026-06-04 14:17 CT - SSR allowed-host gate

- Added an Express SSR host gate that validates `Host` and `X-Forwarded-Host` against approved static host families or the resolved local/runtime `site-config` host set before Angular SSR handles the request.
- Angular SSR now receives the exact validated host set for each request, allowing newly published runtime-backed domains without per-domain `angular.json` hotfixes and without using a wildcard Angular host allowlist.
- Added SSR smoke coverage for `erosbarajas.com`, runtime aliases, unknown custom hosts, test-host `draftDomain` preview, and production hosts ignoring cross-draft `draftDomain` query params.
