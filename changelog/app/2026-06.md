# App Changelog - 2026-06

## 2026-06-04 16:10 CT - Angora combo CSS first paint

- Fixed a first-paint race where `RuntimeService` could ask Angora to generate authored component classes before `AngoraCombosService` had applied the latest `ConfigStoreService` combo payload, so combo-only classes such as `sectionTitle` were filtered and the Eros draft could reveal with stale title color.
- `AngoraCombosService` now synchronizes draft combos from the store before explicit class updates, splits authored class strings into individual tokens, and checks rendered combo `__COM_*` rules before reporting CSS-ready.
- `RuntimeService` now passes rendered classes into loading-curtain readiness and keeps the existing timeout behavior resilient with a timer fallback around frame waits.
- Validation passed: focused `AngoraCombosService` + `RuntimeService` Karma suites 36/36 with Edge as `ChromeHeadless`, `npm run build`, and local SSR first-paint QA for `erosbarajas.com` on desktop/mobile.
